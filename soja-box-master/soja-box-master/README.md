## The soja box is a useful tools for python.

## Usage
### soja_md5

```python
import soja_md5 as sj

# 获取字符串md5
print(sj.md5("Hello world"))

# 获取文件md5
print(sj.md5('./README.md', mode='F'))

# 获取文件sha1
print(sj.get_file_sha1('./README.md'))
```
